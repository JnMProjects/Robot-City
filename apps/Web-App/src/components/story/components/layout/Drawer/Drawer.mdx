# Drawer

<Canvas of={DrawerStory.Default} />

The Drawer component is used to display a modal that slides in from the side of the screen. It is used to display additional content or actions that are not part of the main view.

### Pre Styled!!!

## Usage

```tsx
import {
    Drawer,
    DrawerTrigger,
    DrawerContent,
    DrawerHeader,
    DrawerTitle,
    DrawerDescription,
    DrawerClose,
    DrawerFooter
} from "@layout/.";

<Drawer.Wrapper>
    <Drawer.Trigger>
        {/* Trigger goes here */}
    </Drawer.Trigger>
    <Drawer.Content>
        <Drawer.Header>
            <Drawer.Title>
                {/* Title goes here // Text Possible */} 
            </Drawer.Title>
            <Drawer.Description>
                {/* Description goes here // Text Possible */}
            </Drawer.Description>
            <Drawer.Close>
                {/* Close button goes here // Also Text Possible */}
            </Drawer.Close>
        </Drawer.Header>
        <Drawer.Footer>
            {/* Footer goes here // Text Only */}
        </Drawer.Footer>
    </Drawer.Content>
</Drawer.Wrapper>
```

Important to note is that the Buttons Are Styled in Text Only.
