# Skeleton

<Skeleton>HEHEHE</Skeleton>

The Skeleton component is used to display a loading state for a component. It is used to indicate to the user that the content is loading and will be displayed shortly.

### Pre Styled!!!

<Primary />

## Usage

```tsx
import { Skeleton } from "@layout/Skeleton"

export const MyComponent = () => {
  return <Skeleton>{children}</Skeleton>
}
```

### Tips:

You can also Use the Skeleton by adding the Tailwind Animation:

```tsx
<div className="animate-skeleton">// Some Stuff</div>
```
